# Kchecker  - 漢字未習チェックツール

CSVファイルを読み込み、指定した学年までに習っていない漢字を瞬時に抽出・可視化するWebアプリケーションです。

## 特徴

- **高速チェック**: アップロードしたCSV内の漢字を正規表現とデータベースで高速に判定。
- **学年別フィルタリング**: 小学校1年生から6年生まで、ターゲットとする学年を切り替えてチェック可能。
- **プライバシー配慮**: ファイルの解析はブラウザ上で行われるため、データがサーバーに送信されることはありません。

## 使い方

1. **学年を選択**: 画面上部のピルボタンから、判定の基準とする学年を選択します。
2. **CSVを選択**: 「CSVを選択」ボタンから、チェックしたいファイルをアップロードします。
3. **結果を確認**: 未習漢字が含まれる問題がリストアップされ、該当する漢字と習う学年が表示されます。

## 環境

- **Frontend**: HTML5, Vanilla CSS, JavaScript (ES6+)
- **Build Tool**: Vite
- **Library**: PapaParse (CSV解析)
- **Deployment**: Vercel
